1. **Type Parameter Addition**
   - `list` → `list[str]`, `list[Dict[str, Any]]`
   - `dict` → `dict[str, int]`, `dict[str, Any]`
   - `Dict` → `Dict[str, Any]` with proper generics

2. **Return Type Annotations**
   - Added `-> None` to all `__init__` methods
   - Added `-> Dict[str, Any]` to return statements
   - Fixed implicit Optional types to `Optional[T]` or `T | None`

3. **Collection Type Conversions**
   - `Sequence[str]` → `list[str]` (when mutations used)
   - `Collection[str]` → `list[str]` (when indexing/mutations)

4. **Import Fixes**
   - Changed absolute imports `from src.*` → relative `from . import`
   - Added missing dataclass imports
   - Added TYPE_CHECKING blocks for circular imports

5. **Type Casting**
   - Used `cast()` for Dict[str, Any] access
   - Proper int/float conversion handling
   - Optional[T] null checks
